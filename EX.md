
# Demo

```bash
    
    git help --all

    git add .
    git status

    git restore --staged .

    git add -all # git add -A

    git status --short

    # Commit without Stage
    git commit -a -m "Update EX.md [bash]"
```

## Short status flags
`??` - Untracked files
`A` - Files added to stage
`M` - Modified files
`D` - Deleted files
